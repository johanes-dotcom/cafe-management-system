# ☕ CafePOS

CafePOS adalah aplikasi Point of Sale (POS) berbasis web yang dikembangkan menggunakan **React**, **Vite**, dan **Tailwind CSS**. Aplikasi ini dirancang untuk membantu operasional kafe, mulai dari pengelolaan menu, transaksi, hingga manajemen pengguna.

## ✨ Fitur

- 🔐 Login Authentication
- 👤 Manajemen Pengguna
- 📋 Manajemen Menu
- 🛒 Transaksi Penjualan
- 💳 Pembayaran
- 📊 Dashboard Admin
- 🧾 Riwayat Transaksi
- 📱 Responsive Design

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- React Router
- Axios

## 🚀 Instalasi

Clone repository

```bash
git clone https://github.com/johanes-dotcom/kafe-pos.git
```

Masuk ke folder project

```bash
cd kafe-pos
```

Install dependencies

```bash
npm install
```

Jalankan project

```bash
npm run dev
```

Build untuk production

```bash
npm run build
```

Preview hasil build

```bash
npm run preview
```
